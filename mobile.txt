سأبدأ بحل وشرح الموجود، ثم أعطيك 40 سؤال إضافي مشابه بنفس الأسلوب.

حل الأسئلة الموجودة مع الشرح
1️⃣ What is the correct command to create a new React Native project?
✅ Answer: expo init myReactNativeApp
💡 شرح: في المشاريع الحديثة، أسهل طريقة لبدء مشروع React Native هي استخدام Expo CLI، والأمر هو expo init متبوع باسم المشروع.

2️⃣ What command is used to start the React Local Development Server?
✅ Answer: npm start
💡 شرح: هذا الأمر يقوم بتشغيل Metro Bundler لبدء المشروع في وضع التطوير.

3️⃣ What is the default local host port used by a React development server?
✅ Answer: 3000
💡 شرح: سيرفر التطوير في React يعمل افتراضيًا على port 3000، إلا إذا كان محجوزًا.

4️⃣ What keyword creates a constant in JavaScript?
✅ Answer: const
💡 شرح: الكلمة const تُستخدم لتعريف متغير ثابت القيمة.

5️⃣ Select the apps that use React Native:
✅ Answer: All of the above
💡 شرح: Instagram، Facebook، وSkype جميعها تستخدم React Native.

6️⃣ What tool does React use to compile JSX?
✅ Answer: Babel
💡 شرح: Babel يقوم بتحويل JSX إلى JavaScript عادي يمكن للمتصفح فهمه.

7️⃣ What is the correct way to import a component from React Native?
✅ Answer: import ComponentName from 'react-native'
💡 شرح: يجب أن تستورد المكوّن من مكتبة react-native بالاسم الصحيح.

8️⃣ How do you style your React Native component?
✅ Answer: StyleSheet.create
💡 شرح: React Native يستخدم StyleSheet.create() لإنشاء كائن أنماط (styles).

9️⃣ Which property is used to change the direction of rows/columns in React Native?
✅ Answer: flexDirection
💡 شرح: flexDirection يمكن أن تكون row أو column لتحديد اتجاه العناصر.
